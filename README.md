# Titanic Data Preprocessing
  
The goal is to prepare and clean the Titanic dataset before applying machine learning models.

## Steps
1. Data exploration and handling missing values  
2. Encoding categorical variables  
3. Normalizing data using MinMaxScaler  
4. Splitting data into training and testing sets (optional)

## Tools Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn
